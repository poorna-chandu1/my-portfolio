About me

I am a Computer Science and Engineering student with a strong passion for developing innovative solutions, particularly in web development and cybersecurity. My experience includes the development of a web-based Intrusion Detection System (IDS) to predict Distributed Denial of Service (DDoS) attack types using machine learning. In this project, I implemented data preprocessing, feature extraction, and encoding techniques  to prepare real-world network traffic data  and trained machine learning models, specifically Random Forest and XG Boost, on the UNSW-NB15 dataset. I also developed interactive visualizations and user interfaces for service providers and remote users. This full-stack solution was built using the Django (Python) backend and HTML/CSS/JS frontend. Beyond this, I have hands-on experience in creating personal portfolio websites using HTML, CSS, and JavaScript. I am committed to applying my technical skills to create impactful and user-friendly applications.
